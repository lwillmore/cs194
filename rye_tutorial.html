I do have a working installation. Here is what you should do:

 

unzip the lib.tar.gz file into a lib directory in your home.

unzip the include.tar.gz file into a include directory in your home.

unzip the leveldb.tar.gz file into a leveldb directory in your home.

 

Modify the Makefile.config in caffe as follows:

 

INCLUDE_DIRS := $(PYTHON_INCLUDE) /usr/local/include <YOUR HOME DIRECTORY>/include <YOUR HOME DIRECTORY>/leveldb/include
LIBRARY_DIRS := $(PYTHON_LIB) /usr/local/lib /usr/lib <YOUR HOME DIRECTORY>/lib <YOUR HOME DIRECTORY>/leveldb
Also, on your shell, do
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:<YOUR HOME DIRECTORY>/lib:<YOUR HOME DIRECTORY>/leveldb:/usr/local/cuda/lib64:/usr/local/cuda/lib
Now you should be able to build caffe. Note this will only work on rye if you want to build the GPU versions, since the CUDA libraries are not present on corn machines.
